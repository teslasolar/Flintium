Architecture

The site is data-driven. Page navigation is rendered from pages/assets/manifest.json, every UDT type lives in its own file under pages/data/udts/, and example tag instances live under pages/data/instances/. Add a new UDT or instance by dropping a JSON file — no HTML changes required.
